Проверяем сессию...

Уход за растениями без лишней архитектурной магии

PlantCare помогает вести растения, календарь ухода и погодные подсказки

Backend работает на Django REST Framework и PostgreSQL, frontend — обычные статические HTML/CSS/JS-файлы, которые отдает nginx без Node.js в контейнере.
